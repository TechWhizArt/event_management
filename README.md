# 🎉 Event Management System

A modern and responsive **Event Management Web Application** built using **React.js**. The platform allows users to discover events, register, and manage their accounts through an intuitive user interface.

---

## 🚀 Features

- 🔐 User Authentication
  - Login
  - Signup
  - Forgot Password
  - Reset Password

- 🎫 Event Management
  - Browse upcoming events
  - View event details
  - Register for events

- 🎨 Modern UI
  - Responsive Design
  - Interactive Cards
  - Attractive Landing Page
  - Clean Navigation

- ⚡ React Features
  - React Router
  - Component-Based Architecture
  - State Management using Hooks
  - Form Validation

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/event_management.git
```

Go inside the project

```bash
cd event_management
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
